概念升级

发表于 2017-02-17 | | 阅读次数

概念升级

这些天，对知识的强烈的渴求，简直是到了如饥似渴的地步。似乎阅读、学习新技术已经逐渐成了自己的刚需！我是如何做到的呢？最重要的是对一个个概念的升级，让这些概念在电脑里产生沟回，深入理解了这些概念想不提高也难啊。

注意力是最宝贵的财富

一个人能力最重要，提升能力的过程，我们称之为“成长”，成长是实现财富自由的根本。因此，我们要将注意力用在自己的成长上。你的注意力很少，真的很少，一天下来，能够集中起来有产出的注意力，弄不好往往只有两三个小时而已。大多数注意力可能都用在了凑热闹、随大流、操别人的心去了。

钱不是最重要的，因为它可以再生；时间也不是最重要的，因为它本质上不属于你。你的注意力才是你拥有的最重要、最宝贵的资源。你的注意力可以自己做主，除非是你自己放弃。必须把最宝贵的注意力全部放在自己的身上、自己的成长上。

什么是坚持

人的大脑有遗忘痛苦的能力。越是痛苦的事情大脑遗忘的越快。当人的大脑发现做某一件事情是需要“坚持”的，就会给它赋予了太多的痛苦和坚信，你的大脑自然会想方设法帮你抹掉这部分记忆。

在做任何事情开始之前，需要赋予这件事情重大的意义。这个重大意义有两层意义，一个是关于，如果我不做这件事，会产生什么不良的后果？会变成什么样子？另外一个层意义是，如果我做好了这件事情，会产生什么积极的影响？对自己造成什么好的影响？或者是成长？因此，以后在做任何事情的时候，就不会想到要坚持，要有毅力，只要给这件事情赋予重大的意义就可以了。

以后，在我的操作系统里，没有“坚持”这个词了。

我们是如何错过的

对于不明白的或者自己以为自己明白的知识点，别人在讲的时候，我们会以为这不关我的事儿就不认真听不去理解；他讲的我都明白，写得太好了，说的太对了，然后不去践行。

对于一个概念、一些技术、好的文章，我们总是先“收藏”或者“分享”，然后就再也不去看不去学了。我们就是这样一次次错过的，错过了升级自己的机会！

正确本身的价值

一直以来，自己都理解错了什么是正确，总是在想办法证明自己是正确的。比如证明自己的某个想法，某件事是正确的。昨天才知道，正确本身是毫无价值的。你是正确的，然并卵！

当你是正确的，与此同时，别人也都是正确的，那“正确”本身的价值其实并不大。你是错误的，别人都是正确的，那倒是很可怕。当你是正确的，可别人都是错误的，这时候，“你的正确”才具备很大的价值。通过李笑来老师从正确程度与不被认可程度的坐标图可以看出：你正确的程度越大，与此同时，不认同你的人越多，这样的时候，价值才很大。你很正确，但与此同时，所有人都很正确，那么，价值其实可能等于零。所以，当我确认自己是绝对正确的时候，我为什么一定要活得别人的认可呢？

什么是价值观

昨天确确实实体会到了一个道理，那就是当人在做某一件事情的时候，如果没有全神贯注，是不能把这件事情真正做好的。比如，昨天早上在更新“价值观”概念的时候，周围环境嘈杂，静不下心来。一遍读完，似乎没有以前那么大的收获与震撼力，于是晚上重新读一遍，这种思想上提升感觉依然让人很爽。所以，以后在做事情的时候，尤其是利于自己成长的事情的时候，一定要分配足够的注意力去做这件事。

所谓的成功，无非就是解答题高手做对了选择题

很多人会把注意力放到如何提高解答题能力上。但是很少有人会刻意打磨自己做选择题的能力。因为大多数人没有养成正确有效的价值观。所谓的价值观，最通俗、最有效的定义无非是：

知道什么好，什么更好，什么最好。

因此打磨自己的价值观，就是要让自己知道什么好，什么更好，什么最好。知道了什么好，什么更好，什么最好，也就知道了什么差，什么更差，什么最差。
